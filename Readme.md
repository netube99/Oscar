# Oscar 物联网开发板
基于GD32F130C8T6为主控设计的物联网开发板，板载丰富的芯片/模块，涵盖了UART/SPI/I2C/ADC等外设接口，通过学习使用Oscar 开发板，用户可快速入门GD32F130系列芯片的基础操作；
* ESP8285 Wi-Fi 模块
* CP2102 USB串口
* 电压电流测量电路
* 128kb 串行Flash
* 3.2寸SPI串行TFT 液晶触摸屏
* 2个按键，可唤醒主控
* 小型贴片蜂鸣器
* USB供电过流保护设计
