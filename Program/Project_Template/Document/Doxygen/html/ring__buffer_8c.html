详细描述

简易环形缓冲的实现

可作为单片机串口的环形缓冲/软FIFO，提升串口数据传输能力； 可以向环形缓冲区快速的储存/读取数据，并且无需手动记录数据接收量即可准确的读出所有数据； 无需手动清空数据缓存区，只要将上次接收的数据读取出来，缓冲区即可准备好接收下一段数据； 节省了手动清空普通缓存区的时间，能够提升串口程序的运行效率；
